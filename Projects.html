Grenadier-465: Discontinued

Grenadier-521 is a space exploration game where the player controls every detailed ship command themselves.
As you can see the player can control the direction, rotation, and speed of the ship. Everything is designed on a ds_grid with realistic proportions.
This project required me to design a 3D space within GameMaker Studio 2, a 2D platform.
I learned a lot during this project and visited many ideas to make it work such as skybox mapping other sectors onto a drawn map, limited visibility, and randomly generated 'solar' systems.
Currently in this video, more than 10,000 stars are being randomly generated and drawn constantly on the screen since I left the project in a testing state.
There is also a lot of math behind the scenes, in order to accurately generate sections of a galaxy.
The ridiculous X, Y, and Z values are attributed to the current scale of the game being galactic.
I also demonstrate my organizational skills with this project, keeping track of the systems and scale. (Even if the scale of the game got out of hand, it was organized!) Unfortunately, a space game with realistic proportions gets out of hand very quickly and the project was put aside indefinitely.
This idea is dear to me and I continue to hope one day to be able to revisit it.

AIlex: Twitch Integrated OpenAI

AIlex is the first 'AI' chatbot I designed and rigged using OpenAI's Language Model, Google API, VTube Studio, and Twitch integration.
The entire project runs in one Python .exe (with VTube Studio and whatever recording or streaming software you use).
This particular program uses Twitch Chat commands or even just messages to format and send prompts to a language model and then reads out the model using Google's Text-to-Speech API.
The audio is then sent to a virtual sound device and monitored by VTube Studio.
The entire project is over 700 lines of code, using multithreading, writing and reading json, and much more.
Future similar projects even had Discord integration.
Highly customizable and optimized, this project demonstrates my knowledge in communicating with multiple different API's, and my organizational skills.
